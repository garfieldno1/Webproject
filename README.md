# README

1.เริ่มแรกให้ทำการดาวน์โหลดไฟล์ zip จาก GitHub

2.เมื่อดาวน์โหลดสำเร็จให้ทำการเปิดไฟล์ขึ้นมาโดยแยกเป็นสองส่วนคือ
 -Folderใน vs code (Front-end, Back-end)
 -ไฟล์ใน MYSQL Workbench (data base)

3.ทำการ run file data base เพื่อให้ระบบสามารถเข้าถึงและใช้งานข้อมูลในฐานข้อมูลได้

4.สร้างไฟล์. env ในไฟล์ folder sec2_gr3_ws_src แล้วทำการใส่ข้อมูลดังนี้
 -DB_HOST = localhost
-DB_USER = root
-DB_NAME = sec2_gr3_database
-DB_PASS = (ใช้รหัสของแต่ละเครื่อง)
-PORT = 8000

5.ทำการลง node_modules โดยใช้คำสั่ง `npm install ` ใน terminal ของ folder sec2_gr3_ws_src

6.ใน folder sec2_gr3_fe_src ก็ทำเช่นกันเหมือนกับข้อ 5 แต่ทำใน folder sec_gr3_fe_src

7.ให้ทำการพิมพ์ npm start ในทั้งสอง Terminal

8.หน้าเว็บจะเด้งให้ทันทีหากไม่เด้งให้พิมพ์ `http://localhost:3000/` ในแถบ URL เพื่อเป็นการเริ่มใช้งาน web 

9.ระบบจะทำการแสดงหน้า **Landing Page** 

## Landing Page
ในหน้านี้เราจะประกอบไปด้วย **Nav Bar** ที่มีลิงค์ไปยังหน้าต่างๆได้แก่:Facility, Tournament, Booking Check, Contact (สามารถลองกดเพื่อดูรายละเอียดและการแสดงผลของแต่ละหน้า ซึ่งแต่ละหน้าจะมีรายละเอียดต่างๆ และรูปภาพประกอบ) นอกจากนี้ในหน้า Landing page ของพวกเรายังมีการใช้ web service สาธารณะ คือ สภาพอากาศของสนามกีฬา(Today weather)

## Contact Page 
ในหน้านี้จะมีรูปภาพและรายชื่อของสมาชิกกลุ่มและลิงค์เชื่อมต่อไปยังสื่อ social

## Search Page 
ในหน้านี้เราสามารถค้นหาข้อมูลที่ต้องการโดยการกรอกข้อมูลที่ต้องการเช่น ประเภทบริการ(Service), ชื่อบริการ(Name), ประเภทกีฬา(Sport Type), วันที่(Date) ลงใน input box หลังจากคลิกปุ่ม Search ระบบจะนำไปยังหน้า Search result และแสดงผลข้อมูลที่ตรงกับที่ผู้ใช้คนหาโดยดึงมาจากฐานข้อมูลของระบบ

## Detail page 
ในหน้านี้ระบบจะแสดงรายละเอียดบริการของพวกเราทั้งหมดโดยอ้างอิงจากฐานข้อมูล(ต้องกดที่รูปภาพเพื่อแสดงหน้า detail page)

## Manage User Page
หน้านี้ระบบจะแสดงข้อมูลของ**User** ได้แก่ User_Id, User_Name, Email, BirthDate, Role



## ฟังก์ชัน Search 
1.	เราจะมีช่องสำหรับการ search ให้กรอกข้อมูลที่ต้องการ Search ได้เลย (ให้เจาะจงสิ่งที่ต้องการค้นหา)
2.	หลังจากรอกเสร็จแล้วให้ทำการกดตรงแว่นขยายเพื่อเป็นการ Search 

## ฟังก์ชัน Edit
1.	ในแต่ละข้อมูล **User** จะมีปุ่ม **Edit** สำหรับการแก้ไขข้อมูล
2.	ให้กดที่ปุ่ม **Edit** เพื่อเข้าไปแก้ไขข้อมูลที่ต้องการเปลี่ยนแปลง
3.	กรอกข้อมูลที่ต้องการแก้ไข
4.	เมื่อแก้ไขข้อมูลเสร็จแล้ว ให้กดปุ่ม **Confirm** ระบบจะทำการอัปเดตข้อมูลในระบบ

## ฟังก์ชัน Delete
1.	เมื่อกดปุ่ม **Delete** ระบบจะแสดงข้อความเพื่อยืนยันการลบข้อมูล โดยจะแสดงข้อความถามว่าจะลบข้อมูลนี้จริงๆ หรือไม่
2.	หากต้องการลบจริง ให้กดปุ่ม **Confirm** ระบบจะทำการลบข้อมูล **Facility** นั้นออกจากระบบ

## ฟังก์ชัน Add User
1.	เมื่อกดปุ่ม **Add user**ระบบจะแสดงฟอร์มสำหรับการ**Add user**
2.	กรอกแบบฟอร์มให้ครับแล้วกด**confirm add**
3.	ระบบจะทำการAdd user

## ฟังก์ชัน Add Admin user
1.	เมื่อกดปุ่ม **Add admin**ระบบจะแสดงฟอร์มสำหรับการ**Add admin**
2.	กรอกแบบฟอร์มให้ครับแล้วกด**confirm add admin**
3.	ระบบจะทำการAdd admin
